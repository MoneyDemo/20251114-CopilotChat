## 📖 專案簡介

本專案是 Microsoft 365 Copilot Chat 的實踐示範庫，展示如何利用 Copilot 的 AI 能力提升工作效率。專案包含多種實用場景的提示詞（Prompt）範例，涵蓋市場行銷、文件撰寫、翻譯等實務應用。

### 🎯 專案目標

- 提供實用的 Copilot Chat 提示詞範例
- 展示 AI 輔助工作流程的最佳實踐
- 協助使用者快速上手 Microsoft 365 Copilot
- 分享 AI 在日常工作中的應用技巧

---

## ✨ 功能特色

### 🔍 智慧搜尋與研究
- **市場趨勢分析**：自動搜尋並整理特定主題的最新資訊
- **新聞摘要**：快速獲取指定時間範圍內的相關新聞
- **專業語氣調整**：依據使用情境調整回應的語氣風格

### ✍️ 內容創作與編輯
- **部落格文章生成**：根據主題自動撰寫文章
- **內容濃縮**：將長文轉換為重點摘要
- **創意文案**：生成押韻標語（SLOGAN）等創意內容
- **協作功能**：支援多人協同編輯與互動

### 📊 資料處理與整理
- **表格生成**：基於內容自動建立投票表格、統計表等
- **檔案摘要**：快速理解文件核心內容
- **提示教練**：提供更好的提示詞建議

### 🌐 多語言翻譯
- **專業翻譯**：支援多語言到繁體中文的翻譯
- **校稿優化**：檢查語法與用詞，確保符合繁體中文語言習慣
- **多步驟翻譯流程**：模擬專業翻譯社的工作流程
- **Declarative Agent**：自訂專屬翻譯助理

## 🚀 使用指南

### 前置需求

- Microsoft 365 帳號
- 啟用 Copilot 功能的存取權限
- 瀏覽器（建議使用 Edge 或 Chrome）

### 快速開始

1. **存取 Copilot Chat**
   
   前往 [https://m365copilot.com/](https://m365copilot.com/)

2. **選擇適合的提示詞**
   
   瀏覽 `20251114.Demo.md` 檔案，找到符合需求的使用場景

3. **複製並自訂提示詞**
   
   根據實際需求調整提示詞內容

4. **開始對話**
   
   在 Copilot Chat 中貼上提示詞，體驗 AI 輔助工作流程

## 💡 示範案例

### 案例一：市場行銷研究

**使用場景**：行銷人員需要快速了解特定公司的最新動態

**提示詞範例**：
```
我從事市場行銷工作，專注于行銷趨勢的研究。請簡要介紹有關台灣微軟的最近新聞。關注過去 2 個月的 Web 文章。用兩到三段話回答，並使用商務語氣。
```

### 案例二：內容創作

**使用場景**：撰寫部落格文章並優化內容

**提示詞範例**：
```
請幫我撰寫一篇 500 字的部落格文章，主題是「遠端工作的三個好處」。
```

**後續優化**：
```
將這篇文章濃縮成 3 個重點。
```

**加入創意元素**：
```
加上押韻的 SLOGAN
```

### 案例三：多語言翻譯

**使用場景**：將日文文章翻譯成繁體中文

#### 方法一：單純翻譯
```
請將以下日文文章翻譯成繁體中文：
抜けるような青空と鮮やかな色彩
11月の日本は空気が澄み渡り、抜けるような青空が広がり、また色とりどりの紅葉が色を添えて旅行するのに快適で適した季節になります。気温は、特に朝晩は徐々に肌寒くなり、温泉旅行はそんな冷たい空気を感じた時の癒しになるでしょう。クリスマスのライトやイルミネーションのイベントにも注目してみてください。11月に始まるものも多いです。
```

#### 方法二：使用 Declarative Agent
```
依據專業翻譯社流程，協助使用者將各種語言文章翻譯為繁體中文，並於每個步驟回饋詳細結果。回復請使用繁體中文。Agent名稱：翻譯駐守
```

更多詳細範例請參考 [20251114.Demo.md](./20251114.Demo.md)

## 📝 最佳實踐建議

### 撰寫有效提示詞的技巧

1. **明確指定目標**：清楚說明想要達成的目標
2. **提供背景資訊**：說明使用情境與身份角色
3. **設定輸出格式**：指定所需的格式、長度、語氣
4. **使用具體範例**：提供參考內容或格式範例
5. **迭代優化**：根據回應持續調整提示詞

### 提示教練功能

使用「提示教練」功能獲得更好的提示詞建議：
```
詳細摘要檔案
```

## 🤝 協作功能

Copilot Chat 支援多人協同編輯，您可以：

- 透過連結分享對話內容
- 使用 @mention 邀請團隊成員參與
- 基於討論內容建立表格、文件等

**範例**：
```
幫我基於上面總結的三點，建立投票表格
```

## 🙋 常見問題

<details>
<summary><strong>Q: 如何取得 Microsoft 365 Copilot 的存取權限？</strong></summary>

A: 請聯繫貴組織的 IT 管理員，確認是否已啟用 Copilot 功能。個人用戶可參考 Microsoft 官方網站的訂閱方案。
</details>

<details>
<summary><strong>Q: 提示詞可以用英文撰寫嗎？</strong></summary>

A: 可以。Copilot 支援多種語言，但建議使用與期望輸出相同的語言撰寫提示詞，以獲得最佳效果。
</details>

<details>
<summary><strong>Q: 如何改善 Copilot 的回應品質？</strong></summary>

A: 提供更多背景資訊、使用明確的指令、參考本專案的提示詞範例，並利用「提示教練」功能優化您的提示詞。
</details>
